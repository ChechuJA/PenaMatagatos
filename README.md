# Peña Matagatos

Sitio web oficial de Peña Matagatos - Una peña de un pueblo de Cuenca, pequeña y acogedora.

## Sobre el Sitio Web

Este repositorio contiene el código fuente del sitio web de Peña Matagatos, desarrollado con Jekyll y hospedado en GitHub Pages.

## Ver el Sitio Web

Visita nuestro sitio web en: [https://chechuja.github.io/PenaMatagatos](https://chechuja.github.io/PenaMatagatos)

## Desarrollo Local

Para ejecutar el sitio localmente:

```bash
# Instalar dependencias
bundle install

# Ejecutar el servidor local
bundle exec jekyll serve

# El sitio estará disponible en http://localhost:4000
```

## Estructura del Proyecto

- `_config.yml` - Configuración principal del sitio Jekyll
- `index.md` - Página principal
- `about.md` - Página "Sobre Nosotros"
- `events.md` - Página de eventos y actividades
- `contact.md` - Página de contacto
- `Gemfile` - Dependencias de Ruby/Jekyll
